快速发布

如果你想像发微博一样快速更新，可以用“复制模板 + 只改几处字段”的方式：

  1. 复制一个现有文章目录（例如 2023/05/06/弦/），改成新的日期与标题目录。
  2. 只修改这些位置：标题、副标题、发布时间、正文、许可信息。
  3. 在 index.html、archives/index.html、local-search.xml 里各补一条索引信息。

这种方式不需要重新生成站点，直接改 HTML 就能发布，最接近“随手写”。

建议的极简模板

  • 标题：一句短语即可。
  • 正文：3-6 段，每段一个小主题。
  • 日期：用 YYYY-MM-DD 排序最直观。

如果你愿意，我也可以帮你把“快速发布”进一步变成一键脚本或 GitHub Actions。
