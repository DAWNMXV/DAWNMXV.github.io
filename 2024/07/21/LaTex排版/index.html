LaTex排版

Posted on 2024-07-21 Edited on 2026-01-20 Word count in article: 829 Reading time ≈ 3 mins.

来源

最近帮师哥用 LaTeX 排了一篇 Springer Nature 旗下的期刊，第一次用 LaTeX 排版，记录一下遇到的小问题和细节。

准备工作

把图片转成 PDF，再转为 EPS 格式，使用 Inkscape 这个软件，导出格式选择封装的 PostScript，即 EPS。

文档基本设置

[] 里面表示文档类的参数，分别是数学物理格式，双列布局，{} 里面是文档类名称。

宏包

该加的都加进去，只要注意一下不要冲突就行

通讯作者

带上星号

图片插入

上面是单图，下面是多张图片：

使用 subfigure 可以在一个 Fig 里面插入多个子图，自动进行 a，b，c 编号。

公式插入

使用 SimpleTeX 这个软件，很方便

使用 subequations 在一个公式块内插入多个公式，

使用 split 创建多行公式

采用 & 可以指定对齐的位置，用 ~ 当空格占位置

效果

表格插入

通常使用三线表，\toprule, \midrule, \bottomrule

\multirow{2}{*}{Value} 表示“Value”占两行，这样可以做到表格自动排列不合适情况下自己手动排版。

{@{}p{0.17\columnwidth}p{0.08\columnwidth}@{}}，@{} 去掉左右空白，p 设置占用的长度，l 左对齐，c 居中对齐。

引用

对图标之类的使用标签，引用的时候用 \ref{}，

指明参考文献引用文件

参考文献注意多用 IEEE Trans，而不是 Access，这个是水刊，引用多用期刊，个别会议。

页面设置

在 .cls 文件里，双列页面的话，在 \if@iicol 的最后修改数据

主要修改 paperwidth=193mm, paperheight=260mm。如果是别的页面格式对应修改。

总体目标

使用 Overleaf 编译的结果没报 error 就算成功，否则最好把 error 消完，如果只需要编译出来的 PDF 文件可以不管。
